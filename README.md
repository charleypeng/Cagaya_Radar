# cagaya
a radar display monitor benchmark
#### This project is created by Rider using avalonia ui platform

#### To run the project follow the steps
1. download dotnet sdk version 7
2. run as below
```
dotnet restore && dotnet run
```

#### supported platform
* windows
* mac os
* ios
* android
* linux
